lpinfo(8)

Name

lpinfo - show available devices or drivers

Synopsis

lpinfo [ -E ] [ -l ] [ -m ] [ -v ]

Description

lpinfo lists the available devices or drivers known to the CUPS server. One of the -m or -v options must be specified to get any output:
-E
Forces encryption when connecting to the server.
-l
Shows a "long" listing of devices or drivers.
-m
Shows the available printer drivers on the system.
-v
Shows the available printer devices on the system.

Compatibility

The lpinfo command is unique to CUPS.
