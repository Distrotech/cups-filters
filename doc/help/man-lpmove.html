lpmove(8)

Name

lpmove - move a job to a new destination

Synopsis

lpmove [ -E ] job destination

Description

lpmove moves the specified job to destination. job can be the job ID number or the old destination and job ID: 

     lpmove 123 newprinter
     lpmove oldprinter-123 newprinter

The -E option forces encryption when connecting to the server.

Compatibility

The System V version of this command also allows moving of all jobs from one queue to another. This functionality is currently not supported by CUPS.
