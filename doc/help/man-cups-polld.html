cups-polld(8)

Name

cups-polld - cups printer polling daemon

Synopsis

cups-polld address ipp-port interval browse-port

Description

cups-polld polls remote servers for a list of available printers and printer classes every interval seconds. Printer and class information is then broadcast to the localhost interface (127.0.0.1) for reception by cupsd(8).

This program is started automatically by cupsd for each BrowsePoll directive found in the cupsd.conf file.

See Also

cupsd.conf(5), CUPS Software Administrators Manual, http://localhost:631/documentation.html
